## Instruções para rodar o projeto

### Instalar o projeto

É necessário ter o NodeJS instalado na máquina. Para instalar as dependências do projeto, execute o comando:

```sh
  npm install
```

### Rodar os testes

Para rodar os testes, execute o comando:

```sh
  npm test
```
